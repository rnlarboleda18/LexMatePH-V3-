# Python Script Generation & Execution

You have full permission to create and execute new Python scripts in this workspace to accomplish tasks, diagnose issues, or run one-off scripts. 

When you need to execute Python code:
1. Prefer creating a new `.py` script (e.g., in a temporary file or script file) using the `write_to_file` tool instead of attempting complex one-liners via PowerShell or bash streams.
2. Execute the script using the `run_command` tool.
3. Keep the scripts clean and manageable. You can delete them afterwards if they are no longer needed.
