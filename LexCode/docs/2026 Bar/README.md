# 2026 Bar syllabus (source files)

Place **plain text** copies of the official 2026 Bar syllabi here so the labeling script can read them.

- Supported: `.md`, `.txt` (UTF-8), any depth under this folder.
- Not read automatically: PDF/DOCX — export or paste into `.txt` / `.md` first.

The script `scripts/label_flashcard_bar2026_gemini.py` concatenates these files and uses them as context for Gemini.
